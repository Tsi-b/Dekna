# PWA Icons

This directory should contain the following icon sizes for the Progressive Web App:

## Required Icons

- `icon-72x72.png` - 72x72 pixels
- `icon-96x96.png` - 96x96 pixels
- `icon-128x128.png` - 128x128 pixels
- `icon-144x144.png` - 144x144 pixels
- `icon-152x152.png` - 152x152 pixels
- `icon-192x192.png` - 192x192 pixels
- `icon-384x384.png` - 384x384 pixels
- `icon-512x512.png` - 512x512 pixels

## Additional Icons

- `/apple-touch-icon.png` - 180x180 pixels (place in public root)
- `/favicon-32x32.png` - 32x32 pixels (place in public root)
- `/favicon-16x16.png` - 16x16 pixels (place in public root)
- `/favicon.ico` - Multi-size ICO file (place in public root)

## Icon Generation

Use tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## Design Guidelines

- Use the DEKNA brand colors (Gold: #C9A96E, Background: #FAF8F3)
- Include the "7" logo element
- Ensure icons work on both light and dark backgrounds
- Use "maskable" safe zone (80% of canvas for important content)
